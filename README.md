# Watermark killer bot — удаление водяных знаков по токенам

Бот помогает **удалять водяные знаки** с изображений **только при наличии прав/разрешения** (например, вы владелец контента или у вас есть согласие автора).

Оплата — через **токены**: пополняете баланс и оплачиваете обработки.

---

## Бонус новичка
При первом запуске бот начисляет **10 токенов бесплатно** (1 раз на пользователя).  
`/назад`

---

## Возможности
- Удаление водяных знаков с **PNG / JPG / WEBP**
- Показ стоимости **до подтверждения**
`/назад`

---

## Токены и стоимость (без подписок)

| Действие | Стоимость | Что вы получаете | Примечание |
|---|---:|---|---|
| Оценка / предпросмотр | 0 токенов | Анализ и прогноз результата | Без списания |
| Удаление водяного знака | 1 токен | 1 изображение | Списание после подтверждения |
| Бонус новичка | +10 токенов | Стартовые бесплатные токены | 1 раз |

`/назад`

## Как пользоваться (и как вернуться)

1. Введите `/start`  
`/назад`

2. Отправьте изображение в чат  
`/назад`

3. Подтвердите списание **1 токена** и получите результат  
`/назад`

4. Если токены закончились: **Баланс → Пополнить** → выберите вариант пополнения  
`/назад`
